FIJI ForthIsh Java Interpreter

  • FIJI the ForthIsh Java Interpreter is an interactive environment with a procedural syntax similar to the Forth programming language. FIJI can load, introspect, and execute Java classes and methods. FIJI's main use is for unit and component testing of Java. Here's a screenshot of FIJI.
  • FIJI's SourceForge project page is at http://sourceforge.net/projects/fiji/
  • FIJI is part of a larger body of the author's work. You can visit the SoftWoehr Open Source Page to see more open source software by Jack Woehr.
