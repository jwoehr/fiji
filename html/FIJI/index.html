What is FIJI?

Go Back to Table of Contents

FIJI is a forth-like interpretive procedural language which is in reality a Java interpreter. You can load and exercise Java classes interactively in FIJI, or you can compile FIJI programs to do so.

FIJI is an open source project licensed under the GNU Public License.

Where do I get FIJI?

The latest FIJI release is also linked from the FIJI on Github.

How Do I Build FIJI?

A Jakarta Ant build file build.xml is used to build FIJI. Invoke ant as follows:

ant -f ./scripts/build.xml

This will build the default target all which includes generating the FIJI jar file. Try:

ant -f ./scripts/build.xml usage

to see a usage message on all the targets you may invoke when using the supplied build file.
