FIJI Examples

[Go Back to Table of Contents]

The FIJI examples are in subdirectories of the /com/SoftWoehr/FIJI/examples subdirectory. The lists below link to the source code and directories.

To load and try out any of the source, change directory to the respective directory, start fiji and load the code, e.g.

cd examples/fiji/awt
java com.SoftWoehr.desktop.shell.interpreter
hellowin.fiji load

Example Directories and Source Code

  • bmbcon
    • A variety of examples by Roelf Toxopeus posted to his Forthish Approach to Java page.
  • fiji
    • awt .. java.awt examples
      • hellowin.fiji .. open a window and say "hello".
    • io .. java.io examples
      • cat.fiji .. display a file like the Unix 'cat' command.
    • util .. various utilities
      • rstack.fiji .. create a Forth-like return stack for temporary pushing of arguments from the main FIJI object stack.
      • dir.fiji .. yield a directory listing by exploiting the class com.SoftWoehr.util.DirSupport.
